# readthecard

マイナンバーカード読み取りアプリケーション

## 概要

Windows 11で動作するマイナンバーカード読み取りアプリケーションです。
券面事項入力補助AP（Kenhojo AP）を使用して、基本4情報（氏名、住所、生年月日、性別）を読み取り、フルスクリーンで表示します。
VOICEVOX ずんだもんによる音声読み上げ機能も搭載しています。

## 機能

- 4桁暗証番号入力（大きな入力ボックス）
- マイナンバーカードの基本4情報読み取り
- フルスクリーン表示
- VOICEVOX ずんだもんによる読み上げ
- 日本語UI

## 必要要件

### ハードウェア
- PC/SC対応のICカードリーダー
  - 推奨: ACS ACR122U, Sony RC-S380/P, Identiv uTrust
  - ISO 14443 Type Bまたは接触型カードインターフェースをサポート
- マイナンバーカード
  - 券面事項入力補助用の4桁暗証番号が必要

### ソフトウェア
- Windows 11
- Node.js 22以上
- VOICEVOX（音声読み上げ機能を使用する場合）
  - https://voicevox.hiroshiba.jp/ からダウンロード
  - ずんだもんの声で読み上げ

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. VOICEVOXのセットアップ

1. [VOICEVOX公式サイト](https://voicevox.hiroshiba.jp/)からVOICEVOXをダウンロード・インストール
2. VOICEVOXを起動（デフォルトで`http://127.0.0.1:50021`でAPIサーバーが起動）

### 3. アプリケーションの起動

#### 開発モード

```bash
npm run electron:dev
```

#### ビルド

```bash
npm run build
```

## 使用方法

1. アプリケーションを起動
2. 券面事項入力補助AP用の4桁暗証番号を入力
3. 「読み取り開始」をクリック
4. カードリーダーにマイナンバーカードを置く
5. 基本4情報が表示される
6. 「ずんだもんに読み上げてもらう」ボタンで音声読み上げ

## 技術スタック

- **フレームワーク**: Electron + React
- **言語**: TypeScript
- **カードリーダー**: pcsclite (PC/SC バインディング)
- **音声合成**: VOICEVOX ずんだもん
- **ビルドツール**: Vite + electron-builder

## 依存ライブラリ

- React 19.2.0
- Electron 39.2.4
- TypeScript 5.9.3
- Vite 7.2.6
- pcsclite 1.0.1 (PC/SC カードリーダー)

## セキュリティ上の注意

- 暗証番号は画面上でマスク表示されます
- 暗証番号を3回間違えると利用停止になる場合があります
- 読み取った個人情報は適切に管理してください
